## 一、模型背景与目的

### 1.1 核心定位
本模型为**金融机构转型金融业务**提供钢铁行业减碳基准路径参考。服务于银行总行（资产组合对标）和企业客户（龙头企业目标验证）两个维度。

### 1.2 核心方法论
- **双路径耦合**: 自上而下（CGE碳预算约束）+ 自下而上（行业技术路径测算）
- **模型本质**: **"由结果（碳强度目标）倒推过程（技术路径比例）"**，但在Excel呈现上必须严格按照Step 1至Step 8的顺序进行公式化推导
- **闭环逻辑**: 碳强度目标 → 技术路径比例 → 碳排放总量 → 占碳预算比例验证

### 1.3 三个情景定义

| 情景 | 对标基准 | 特征 |
|------|----------|------|
| **积极情景** | NZE/1.5°C路线图 | 激进技术转型，低碳技术快速普及 |
| **适度情景** | CGE 2°C（推荐对标） | 政策稳步推进，技术渐进应用 |
| **保守情景** | NDC政策承诺 | 仅执行已承诺政策，技术渗透缓慢 |

## 二、全局硬性规定（不可违反）

### 2.1 基本规则
1. **禁止硬编码**: 除Step 1和Step 2的基准数据外，所有数值必须由公式生成
2. **年份连续**: 必须展示2024-2060每一年的数据，不得省略
3. **情景梯度**: 所有年份碳强度严格满足 **积极 < 适度 < 保守**
4. **基准年约束**: 2024年碳强度=1.95 tCO₂/t，2025年三情景统一=1.904（自然效应）
5. **单位规范**: 产量（万吨），碳排放（万吨CO₂），碳强度（tCO₂/t）

### Step 4: 流程结构（⚠️ 关键可调参数）

**设定逻辑**: 手动设定短流程占比(%)和氢冶金占比(%)，公式推导长流程占比。

**核心公式**:
```
长流程占比 = 1 - 短流程% - 氢冶金%
长流程产量 = 总产量 × 长流程占比
短流程产量 = 总产量 × 短流程% / 100
氢冶金产量 = 总产量 × 氢冶金% / 100
```

**约束条件**:
- 短流程占比逐年递增，三情景差异化（积极>适度>保守）
- 保守情景2030年短流程≥20%（工信部政策要求）
- 2035年积极>30%，2050年积极>45%
- 2060年积极~60%，适度~40%，保守~25-30%
- 短流程+氢冶金占比之和≤100%

**氢冶金约束**:
- 2024年=1%（约1000万吨），已实现
- 2030年≈2900万吨（占总量~3.4%）
- 2050年0.96-1.12亿吨（占总量14%-16%）
- 2060年积极~19%，适度~10-12%，保守~5-8%

### Step 5: 铁素资源结构（纯公式，无需调整）

**核心公式**:
```
长流程废钢消耗 = 长流程产量 × 30%（恒定）
长流程矿石消耗 = (长流程产量 - 长流程废钢消耗/1.1) × 1.053 × 1.65
短流程废钢消耗 = 短流程产量 × 1.1
氢冶金废钢消耗 = 氢冶金产量 × 50%（恒定）
氢冶金矿石消耗 = (氢冶金产量 - 氢冶金废钢消耗/1.1) × 1.053 × 1.65
废钢消耗总量 = 长流程废钢消耗 + 短流程废钢消耗 + 氢冶金废钢消耗
矿石消耗总量 = 长流程矿石消耗 + 氢冶金矿石消耗
```

**废钢资源量参考**: 崔志峰等(2023)预测2045年前后达峰值约5亿吨/年，2050年4.7-5.2亿吨。

### Step 6: 电力结构（⚠️ 关键可调参数）

**核心公式**:
```
总耗电量 = 长流程产量×450 + 短流程产量×550 + 氢冶金产量×500  (万kWh)
总绿电量 = 总耗电量 × 绿电占比
```

**绿电占比锚点（三情景共用或差异化）**:
- 2030: 35%, 2035: 55%, 2040: 70%, 2050: 85%, 2060: 95%
- 其余年份合理插值，曲线平滑

### Step 7: 低碳技术（⚠️ 关键可调参数）

**核心公式**:
```
累计降碳潜力 = 上年累计 + 当年新增降碳潜力
```

**技术部署时序（参考）**:
```
2024: 干熄焦技术 0.066
2025: 厚料层烧结技术 0.06
2026: 球团废热循环利用 0.0075
2028: 高炉TRT余压发电 0.029
2029: 高炉冲渣水余热回收 0.023
2031: 转炉气体及显热回收 0.0046
2032: 转炉负能炼钢 0.0625
2033: 钢包加盖 0.002
2034: 钢包周转优化 0.003
2035: 废钢预热 0.013
2036: 电源智能化控制 0.024
2037: 电炉连续加料 0.007
2043: 连铸坯热送热装 0.013
2047: 薄板坯连铸 0.13
```

**注意**: 三情景可差异化——积极情景技术普及更快，适度次之，保守最慢。2040年前的曲线要尽量平滑，不要阶跃式下降。

### Step 8: 低碳发展测算表（Waterfall瀑布流逻辑）

**8.1 产量变化**:
```
产量变化碳排放曲线 = 粗钢产量 × 1.95
产量变化降碳潜力 = 基准产量(100509) × 1.95 - 当年曲线
```

**8.2 原料结构优化**:
```
基准年废钢总量 = 38394.44（固定）
原料优化降碳潜力 = ((当年废钢总量 - 38394.44)/1.1) × (1.80 - 0.45)
原料优化碳排放曲线 = 产量曲线 - 原料降碳
```

**8.3 短流程比例提升**:
```
基准年短流程废钢消耗 = 11055.99（固定）
短流程提升降碳潜力 = (当年短流程废钢消耗 - 11055.99) × 1.26
短流程提升碳排放曲线 = 原料曲线 - 短流程降碳
```

**8.4 氢冶金比例提升**:
```
基准年氢冶金矿石消耗 = 952.52（固定）
氢冶金提升降碳潜力 = ((当年氢冶金矿石消耗 - 952.52)/1.65) × 1.26
氢冶金提升碳排放曲线 = 短流程曲线 - 氢冶金降碳
```

**8.5 能源结构(绿电)**:
```
绿电降碳潜力 = 总绿电量 × 0.5306 / 1000  (注意单位: kg→t，万kWh→万t)
能源结构碳排放曲线 = 氢冶金曲线 - 绿电降碳
```

**8.6 低碳工艺技术**:
```
低碳工艺降碳潜力 = 当年粗钢产量 × 当年累计技术降碳潜力
低碳工艺碳排放曲线 = 能源曲线 - 技术降碳
```

**8.7 CCUS（⚠️ 配平工具）**:
```
CCUS捕集量 = 低碳工艺碳排放曲线 × CCUS动态比例
```
- CCUS作为最终配平项，比例需逐年平滑递增
- 2030年前可设为0（三情景统一）
- 2035年后开始规模化
- 2060年比例接近100%

**8.8 最终结果**:
```
碳排放总量 = 低碳工艺碳排放曲线 - CCUS
吨钢碳排放强度 = 碳排放总量 / 粗钢产量
碳强度变化(%) = (当年强度 - 1.95) / 1.95
占碳预算比例 = 碳排放总量 / (全国CO₂排放 × 10000)
```

## 四、CCUS约束逻辑（⚠️ 高优先级注意事项）

### 4.1 百万吨起步
- **绝对禁止**从0跳到千万吨级
- 2030年约束: 保守(200-500万吨) | 适度(500-800万吨) | 积极(800-1200万吨)

### 4.2 贡献率约束
- 2060年CCUS减排量占总减排量(BAU-净排放)约15-25%
- 参考IEA: 累计减排量贡献约16%
- 参考清华ICCSD: 处理残余排放10-15%

### 4.3 绝对值参考（适度情景）
| 年份 | CCUS捕集量(万吨) |
|------|-----------------|
| 2030 | ~500-800 |
| 2035 | ~2,500 |
| 2040 | ~6,000 |
| 2050 | ~12,000 |
| 2060 | ~23,000-25,000 |

### 4.4 天花板
- **死线**: 任何情景2060年CCUS绝对不超过**5.4亿吨/年**（Wang et al., 2025）

## 五、已识别的历史错误与注意事项

### 5.1 碳强度曲线急转弯
- CCUS从0突然跳到大规模时，会导致碳强度曲线出现不合理急拐弯
- **解决**: CCUS比例必须逐年平滑递增，形成S型曲线

### 5.2 原料结构降碳潜力为负
- 废钢消耗总量低于基准年时，原料优化降碳潜力为负数（合理现象）
- 但除产量变化外，其他降碳项不应出现大面积持续为负

### 5.3 2025年碳强度
- 三情景2025年统一为1.904（不含CCUS的自然效应）
- Dashboard中2025积极=1.947，说明2025年可能需要三情景差异化处理
- **2025年吨钢综合能耗不降反升+1.9%**是统计数据事实（产品结构调整+超低排放改造+入炉矿品位下降）

### 5.4 氢冶金起点
- 2024年已占1%（约1000万吨），2030年约2900万吨
- 不能假设2030年才从0开始

### 5.5 短流程政策底线
- 保守情景2030年也必须≥20%（《工业领域碳达峰实施方案》）
- 短流程占比只会逐年增加

## 七、校准方法论

### 7.1 校准目标
Step 8公式输出的碳强度必须精确对齐Dashboard（"汇总对比与可视化看板"Sheet）目标值（误差≤0.5%）。

**⚠️ 核心问题**: 当前V11的Dashboard碳强度值是**硬编码**的（非公式引用情景Sheet），而三个情景Sheet的Step 8公式输出与Dashboard存在显著偏差。

### 7.2 当前偏差现状（V11）

| 年份 | 情景 | Dashboard目标CI | Step 8实际CI | 偏差 |
|------|------|-----------------|--------------|------|
| 2030 | 积极 | 1.7440 | 1.5920 | **-0.152（严重偏低，减排过快）** |
| 2030 | 适度 | 1.7980 | 1.7990 | +0.001（基本对齐✓） |
| 2030 | 保守 | 1.8740 | 1.8425 | -0.032（偏低） |
| 2050 | 积极 | 0.1700 | 0.4435 | +0.274（严重偏高，减排不足） |
| 2050 | 适度 | 0.4000 | 0.5371 | +0.137（偏高） |
| 2050 | 保守 | 0.7500 | 0.8000 | +0.050（偏高） |
| 2060 | 积极 | 0.0100 | 0.0040 | -0.006（偏低） |
| 2060 | 适度 | 0.0500 | 0.0060 | -0.044（严重偏低） |
| 2060 | 保守 | 0.1200 | 0.0080 | -0.112（严重偏低） |

**偏差规律总结**:
- **积极情景**: 2030年前减排过快（参数过于激进），2035-2050年间减排不足，2060年又过度依赖CCUS导致偏低
- **适度情景**: 2030年前基本对齐，2035年后减排不足（CCUS起步太晚/太小），2060年严重偏低
- **保守情景**: 整体减排不足，后期CCUS配平不够

### 7.3 校准参数优先级
1. **Step 4**: 短流程占比 + 氢冶金占比（主要调节手段，影响最大）
2. **Step 6**: 绿电占比（辅助调节，影响电力降碳）
3. **Step 7**: 低碳技术部署时序与幅度（辅助调节，影响累计降碳）
4. **CCUS动态比例**: 最终配平项（但必须符合第四章CCUS约束逻辑）

### 7.4 校准步骤
1. 固定Step 1-3不变（基准年参数、全国碳预算、粗钢产量锚点均不调整）
2. 从Dashboard目标倒推Step 8最终碳排放总量 = 目标碳强度 × 粗钢产量
3. 不含CCUS的碳排放曲线 = 最终碳排放 / (1 - CCUS比例)
4. 调整Step 4流程结构使早期(2025-2035)碳强度对齐（积极情景需放缓，保守情景可微调）
5. 调整Step 6/7使中期(2035-2050)碳强度对齐
6. CCUS比例配平使2060年碳强度精确对齐（注意S型曲线约束）
7. 逐年验证：Step 8输出CI vs Dashboard目标CI，确保全部年份误差≤0.5%
8. 验证情景梯度：任何年份满足 积极 < 适度 < 保守
9. 验证CCUS约束：百万吨起步、贡献率15-25%、2060年不超过5.4亿吨
