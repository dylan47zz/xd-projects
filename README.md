## 📐 技术要点

### V12 Step 8 公式重构 (LMDI分解法)

旧公式存在的问题：用绝对量计算降碳潜力，当总产量下降速度快于技术占比提升速度时，技术降碳潜力出现负数，逻辑错误——把"规模效应"混入了"技术效应"。

**修正公式**（确保技术项永远为正）:

```
8.1 产量变化降碳潜力 = (基准产量 - 当年产量) × 基准碳强度1.95
8.2 原料优化降碳潜力 = 长流程产量 × (当年吨钢耗废钢 - 基准吨钢耗废钢) × 铁水碳强度1.26
8.3 短流程降碳潜力 = 当年产量 × (当年短流程占比 - 10%) × (长流程强度1.80 - 短流程强度0.45)
8.4 氢冶金降碳潜力 = 当年产量 × (当年氢冶金占比 - 1%) × (长流程强度1.80 - 氢冶金强度1.65)
8.5 绿电降碳潜力 = 当年产量 × 绿电占比变化 × 电网排放因子
8.6 低碳工艺降碳潜力 = 当年产量 × (基准工艺强度 - 当年工艺强度)
8.7 CCUS配平项 = (基准排放 - 8.1~8.6减排量) - (当年产量 × 目标碳强度)
```

### CCUS配平核心约束
- CCUS只能减少排放（比例≥0）
- 适度情景2060年CCUS占比目标：**10%-15%**（V12达到15.92%，接近合理）
- 配平公式：CCUS比例 = 1 - 目标CI × P / curve_tech
- 限幅：0 ≤ CCUS比例 ≤ 0.98

### 三情景逻辑匹配原则
- **产量**: 保守(高) ≥ 适度(中) ≥ 积极(低)
- **技术渗透率**: 积极(激进) ≥ 适度(中等) ≥ 保守(保守)
- 产量越高 → 技术渗透率越保守 → 碳强度下降越慢 → 符合NDC保守路径
